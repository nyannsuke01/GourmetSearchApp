# GourmetSearch

近くのグルメ情報を取得しMap表示できるアプリ
xcode １１.2　Swift5.0 
ライブラリ　Alamofire SwiftyJSON SDWebImage 

SimpleBrowser.xcodeproj を Xcode で開いて、Yahoo!ローカルサーチAPIのアプリケーションIDを設定すれば、そのまま実行することができます。
